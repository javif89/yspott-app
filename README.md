# yspott-app

Initial code for yspott made at mangohacks.

Running on vuejs using [vue-apollo](https://vue-apollo.netlify.com/) and 8base back-end.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests

npm run test
```
